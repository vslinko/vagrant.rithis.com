Virtual Symfony2 Development Environment

Based on Vagrant, VirtualBox, Debian, and Dotdeb

Installation

1. Download and install VirtualBox

2. Download and install Vagrant

3. Create file named Vagrantfile in your project root with that content:

Vagrant::Config.run do |config|
    config.vm.box = "symfony"
    config.vm.box_url = "http://vagrant.rithis.com/symfony.box"
    config.vm.network :hostonly, "192.168.33.10"
end

You can change IP address of your Vagrant virtual machine, but be careful.

If you're using OS X or Linux you can use nfs instead of VirtualBox shared folders for best performance:

Vagrant::Config.run do |config|
    config.vm.box = "symfony"
    config.vm.box_url = "http://vagrant.rithis.com/symfony.box"
    config.vm.network :hostonly, "192.168.33.10"
    config.vm.share_folder("v-root", "/vagrant", ".", :nfs => true)
end

4. Run shell command vagrant up from your project root directory

5. Open http://192.168.33.10 in your browser

If you're already using database in your project, you need to update database schema.

nginx php5-apc php5-cli php5-curl php5-fpm php5-gd php5-intl php5-mcrypt php5-pgsql php5-sqlite php5-xdebug postgresql
